	格式 
	zafu_ + 指令
	
	指令		功能
	run		启动电机，功能同G2
	stop	停车
	g2s		G2短按
	g2l		G2长按
	g3s		G3短按
	g3l		G3长按
	g4s		G4短按
	g4l		G4长按
	g5s		G5短按
	g5l		G5长按
	d+数字	修改电机占空比，范围50到100
	rj + 符号 + 数字 + ;		设置经度基准		
	rw + 符号 + 数字 + ;		设置纬度基准
	kpd + 数字 + ;			设置直道Kp
	kdd + 数字 + ;			设置直道Kd
	kpc + 数字 + ;			设置弯道Kp
	kdc + 数字 + ;			设置弯道Kd
	p+点位 + rj+符号 + 数字		设置点位的经度基准（点位顺序为1234）
	p+点位 + rw + 符号 + 数字		设置点位的纬度基准
